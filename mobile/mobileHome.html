My name is Brandon Kervin. I am 20 years old and I am a self-taught programmer. I've always had an interest in computers my whole life, be it me trying to hack my xbox as a 7th grader (with success) or just simply trying to understand how a game works by reading save files or even the source code when possible. I took an html course in high-school and instantly knew what I wanted to do with my life. Every since that course, I have had a growing passion for programming and know for sure that this is what I want to be doing for the rest of my life.

My languages/markups include:
  HTML (fluent)
  CSS (fluent)
  JavaScript (adequate)
  Java (subpar)
  C++ (subpar)
  PHP (pretty bad)

Along with all of that, my JavaScript knowledge also includes frameworks like AngularJS, React, JQuery, and Ember. I work very well with JSON and AJAX as well. Expect to see examples of all of these in future updates that get uploaded to the content page.
